# Annotation Strength Evidence

1. YouTube distinguishes creator-uploaded subtitles/captions and automatic captions:
   - https://support.google.com/youtube/answer/2734698
   - https://support.google.com/youtube/answer/6373554
2. `yt-dlp` exposes separate subtitle options (`--write-subs`, `--write-auto-subs`, `--list-subs`) so manual tracks can be enforced:
   - https://github.com/yt-dlp/yt-dlp#subtitle-options
3. TED explains how subtitles are created and reviewed, and provides translator workflow details:
   - https://support.ted.com/hc/en-us/articles/206399647-How-are-TED-Talk-subtitles-created
   - https://www.ted.com/participate/translate/faq
